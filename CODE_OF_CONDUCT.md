# Code de conduite

## Notre engagement

Dans l'intérêt de favoriser un environnement ouvert et accueillant, nous nous engageons à faire de la participation à ce projet une expérience sans harcèlement pour tout le monde.

## Comportements attendus

- Faire preuve d'empathie et de bienveillance
- Respecter les points de vue et expériences différents
- Accepter les critiques constructives avec grâce
- Se concentrer sur ce qui est le mieux pour la communauté

## Comportements inacceptables

- Langage ou images à caractère sexuel
- Trolling, commentaires insultants ou attaques personnelles
- Harcèlement public ou privé
- Publication d'informations privées d'autrui sans permission

## Application

Les comportements abusifs peuvent être signalés en ouvrant une issue privée ou en contactant le mainteneur via GitHub.

Ce code de conduite est adapté du [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
